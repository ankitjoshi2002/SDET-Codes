-> What is a test case??
-> A test case is a work product/deliverable (configurable item CIs) that is written to validate a piece of software under stated conditions or controlled enviourment (test enviourmnt/ test bed) with set of steps(test script) to be performed for a test condition with an expected result with a test data set with an optional precondition(s) and mandatorily a post condition
